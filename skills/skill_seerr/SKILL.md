# Seerr Media Management Skill

Manage your Seerr media server instance for Plex/Emby/Jellyfin.

## Actions

- `check_status` - Check Seerr instance health
- `get_media_status` - Get status by ID or title
- `add_media` - Add new movies/TV shows (needs TMDB ID)
- `remove_media` - Remove media by ID
- `list_pending` - View all pending approval requests
- `approve_media` - Approve pending media by ID
- `decline_media` - Decline pending media by ID
- `get_activity` - Recent activity log (default 10)
