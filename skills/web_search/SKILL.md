# web_search

Search the web for current information using DuckDuckGo. No API key required.

## When to use
- When the user asks about recent events, news, or anything you might not know
- When you need to look up factual information
- When the user explicitly asks you to search the web

## Input
- `query` (string, required): The search query

## Output
Returns a text summary of the top search results, including titles, snippets, and URLs.
